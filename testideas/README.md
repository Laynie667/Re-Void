# testideas/

**The scratchpad.** Half-formed ideas, experiments, "what if", things being crammed into the
live game to see what sticks. Messy on purpose. No format required — dump freely.

Workflow: idea lands here → tried in the live game → if it **works**, it graduates to a clean
blueprint in `../gamedesign/` (or canon in `../loredesign/`); if it doesn't, it stays here as a
record of what didn't land (useful — saves re-trying dead ends in the fresh build).

> Contents here are **gitignored** (except this README). This is local scratch — and remember:
> in the remote/web session, anything here is **not** saved unless committed, so keep the real
> scratchpad on your local machine.

Suggested loose tags inside files: `TRYING`, `WORKS`, `CUT`, `MAYBE`.
