# sideprojects/

**Notes & plans for projects adjacent to the MUD** — tools, companion apps, utilities, anything
that isn't the game itself but supports it or you.

Current threads:
- **Hypno / conditioning app (personal).** A private tool for just you. Recommended route: a
  **PWA** (installable to the home screen, fully offline, no Play Store, no signing) — spirals/
  inductions, TTS or your own audio, mantra/trigger libraries, staged-depth session timers,
  schedules, a journal; with an always-there instant **surface/stop** (the §0 principle). A
  native APK (Flutter/Kotlin) is also possible — final `build apk` would run on your machine.
  **Build it in its own dedicated repo**, not inside Re-Void.

One file per project (e.g. `hypno-app.md`). Capture: the goal, the chosen stack + why, the
feature list, the privacy/safety model, and the build/deploy steps.

> Contents here are **gitignored** (except this README). Working notes live on your local
> machine; each real side project gets its **own repo** when it starts.
