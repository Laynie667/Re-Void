    zone add <name> type=both intimate     — both types

    wear <zone> [clothing description as a full prose paragraph]

    roomzone inscribe/enable <zone>

HOUSING
================================================================================

  Add a character as a resident (can sethome here):
    housing add <character name>           — run while standing in the room

  Remove a resident:
    housing remove <character name>

  List current residents of a room:
    housing residents

  Convert a room to housing (lets player dig off it):
    @typeclass here = typeclasses.housing.HousingRoom
    @py here.db.housing_owner_id = <char_id>
    @py from web.housing.models import HousingPlot; HousingPlot.objects.create(character_id=<char_id>, rooms_total=1, rooms_used=1)

  Grant a character extra room slots:
    @py from web.housing.models import HousingPlot; p = HousingPlot.get_or_none(<char_id>); p.rooms_total += 5; p.save(); me.msg("Done.")
